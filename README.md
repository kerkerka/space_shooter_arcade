# 🚀 Space Shooter Arcade

Космический 2D-шутер на Python с библиотекой Arcade.

---

## 📖 Описание

**Space Shooter Arcade** — это динамичная игра, где вы управляете космическим кораблём и отбиваетесь от волн наступающих врагов.

**Особенности:**
- 🎮 **4 уровня**: 3 уровня с боссами + бесконечный режим
- 👾 **3 типа врагов**: быстрые, средние, танки
- 🎯 **Боссы**: мини-боссы с рывками, финальный босс с призывом подкрепления
- ⚡ **Прокачка оружия**: скорострельность и размер пуль
- 📊 **Рекорды**: сохранение лучших результатов
- 🎨 **Эффекты**: система частиц для выстрелов и взрывов
- 📷 **Камера**: круглая зона видимости с затемнением

---

## 📋 Системные требования

- **ОС**: Windows 7/10/11
- **Python**: 3.8 или выше
- **Оперативная память**: 2 ГБ
- **Место на диске**: 100 МБ

## 🛠️ Установка

### Шаг 1: Установите Python

Если Python не установлен, скачайте его с официального сайта:  
👉 https://www.python.org/downloads/

**Важно:** При установке отметьте галочку ✅ **"Add Python to PATH"**

### Шаг 2: Скачайте игру

```bash
git clone https://github.com/kerkerka/space_shooter_arcade
cd space_shooter_arcade
```

Или скачайте архив и распакуйте в удобную папку.

### Шаг 3: Установите зависимости

Откройте терминал в папке проекта и выполните:

```bash
pip install -r requirements.txt
```

Или вручную:

```bash
pip install arcade==2.6.17 Pillow>=10.0.0
```

## 🎮 Запуск игры

### Из терминала

```bash
python main.py
```

### Из PyCharm

1. Откройте папку проекта в PyCharm
2. Нажмите правой кнопкой на `main.py`
3. Выберите **"Run 'main'"**

## 📊 Игровой процесс

### Уровни

| Уровень | Волны | Босс | Особенности |
|---------|-------|------|-------------|
| 1 | 10 | Мини-босс 1 (100 HP) | Обучение |
| 2 | 10 | Мини-босс 2 (150 HP, рывки) | Сложнее враги |
| 3 | 10 | Финальный босс (300 HP) | Призыв подкрепления |
| 4 | ∞ | Нет | Бесконечный режим |

### Враги

| Тип | Скорость | HP | Урон | Опыт |
|-----|----------|----|----|----|
| 🔸 Быстрый | 8.0 | 1 | 5 | 1 |
| 🟡 Средний | 5.0 | 2 | 10 | 2 |
| 🟣 Танк | 3.0 | 3 | 15 | 3 |
| 🔴 Мини-босс 1 | 3.5 | 100 | 25 | 100 |
| 🔴 Мини-босс 2 | 4.5 | 150 | 25 | 100 |
| 👹 Финальный босс | 2.0 | 300 | 20 | 100 |

### Прокачка оружия

| Волна | Улучшение |
|-------|-----------|
| 3 | Скорострельность ×1.5 |
| 6 | Размер пуль ×1.5 |
| 9 | Скорострельность ×1.5 |
| 10 | Скорострельность ×1.33 (итого ×3.0) |

**Чек-поинт:** После прохождения 9й волны 3го уровня сохраняется прогресс. При смерти вы начнёте с 10й волны!

## 🎨 Ассеты

**Важно:** Для работы игры необходимы графические и звуковые файлы в папке `assets/`!

| Файл | Размер | Описание |
|------|--------|----------|
| `player.png` | 64×64 | Игрок |
| `enemy_weak.png` | 32×32 | Быстрый враг |
| `enemy_medium.png` | 64×64 | Средний враг |
| `enemy_tank.png` | 64×64 | Танк |
| `boss_mini.png` | 96×96 | Мини-босс |
| `boss_final.png` | 128×128 | Финальный босс |
| `bullet.png` | 16×16 | Пуля |
| `particle.png` | 8×8 | Частица |
| `background_level1.png` | 1920×1080 | Фон уровней 1-3 |
| `background_level4.png` | 1920×1080 | Фон уровня 4 |

### Звуки (в папке `assets/sounds/`)

| Файл | Описание |
|------|----------|
| `shoot.wav` | Выстрел |
| `enemy_death.wav` | Смерть врага |
| `player_hit.wav` | Попадание в игрока |
| `player_death.wav` | Смерть игрока |
| `button_click.wav` | Клик по кнопке |

**Без ассетов игра НЕ работает!** Убедитесь, что все файлы присутствуют в папке `assets/`.

## 🐛 Устранение проблем

### Игра не запускается

1. Проверьте версию Python:
   ```bash
   python --version
   ```
   Должна быть 3.8 или выше

2. Переустановите зависимости:
   ```bash
   pip install -r requirements.txt --force-reinstall
   ```

### Ошибка загрузки ассетов

Проверьте, что все файлы присутствуют в папке `assets/`:
- `assets/sprites/` — 10 PNG файлов
- `assets/sounds/` — 5 WAV файлов

### Ошибка `ModuleNotFoundError: No module named 'arcade'`

Установите Arcade:
```bash
pip install arcade==2.6.17
```
