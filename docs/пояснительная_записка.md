## 1. Введение

**Space Shooter Arcade** — 2D-шутер на Python (библиотека Arcade), где игрок отбивается от волн врагов.

**Цель:** Создать игру с изученными технологиями (OOP, SQLite, графика, звук).

---

## 2. Игровой процесс

- **Управление:** WASD — движение, ЛКМ — стрельба
- **4 уровня:** 3 с боссами + бесконечный режим
- **3 типа врагов:** быстрые, средние, танки
- **Боссы:** с рывками и призывом подкрепления
- **Прокачка:** на волнах 3, 6, 9 (скорострельность, размер пуль)
- **Камера:** круглая зона видимости (радиус 500) с затемнением
- **Рекорды:** сохраняются в SQLite

---

## 3. Структура проекта

```
space_shooter_arcade/
├── main.py              # Запуск игры
├── settings.py          # Настройки
├── database.py          # База данных
├── game/
│   ├── player.py        # Игрок
│   ├── enemies.py       # Враги и боссы
│   ├── bullets.py       # Пули
│   ├── particles.py     # Частицы
│   ├── camera.py        # Камера
│   ├── levels.py        # Уровни
│   └── hud.py           # Интерфейс
├── states/
│   ├── menu.py          # Меню
│   ├── game.py          # Игра
│   └── game_over.py     # Конец игры
└── assets/              # Графика и звуки
```

---

## 4. Реализованные технологии

✅ **OOP** — классы: игрок, враги, пули, частицы  
✅ **Графика** — спрайты, анимация, камера  
✅ **Звук** — выстрелы, взрывы, клики  
✅ **SQLite** — сохранение рекордов  
✅ **UI** — меню, HUD, экран победы/поражения  
✅ **Сборка** — .exe через PyInstaller

---

## 5. Запуск

```bash:
pip install -r requirements.txt
python main.py
```

## 6. Заключение
